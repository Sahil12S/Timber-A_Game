# Setting up Game Environment

## Libraries
**SFML**  
Download SFML library, extract it and place contents in a folder named SFML.

## Creating Project Template
1. Create C++ console project.  
2. In project properties, add include directories in C++ (path of SFML/include).  
3. Add path of SFML/lib for Linker->general.  
4. In Debug configuration -> Linker -> Input add "sfml-graphics-d.lib;sfml-window-d.lib;sfml-system-d.lib;sfml-network-d.lib;sfml-audio-d.lib;".  
5. Export project as Template.
